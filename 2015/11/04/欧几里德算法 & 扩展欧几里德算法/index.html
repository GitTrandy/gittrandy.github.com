欧几里德算法 & 扩展欧几里德算法

发表于 2015-11-04   |   分类于 RSA

欧几里德算法是数学里面一个求最大公约数的算法，也被称为辗转相除法。

它的原理基于以下定理：

设 a,b 均为整数

GCD(a,b) = GCD(b,a%b) （GCD表示求两个数的最大公约数）

而扩展欧几里德算法则表述如下：

设 a,b 均为整数

ax + by = GCD(a,b) 总有整数解。

具体数学证明过程我们略过不谈，如果感兴趣可以参考这里。

我们来看这个等式，当a、b不互质的时候，也就是GCD(a,b) != 1。 因为GCD(a,b)是a，b的最大公约数，所以等式左右两边总可以约去公约数，最后会化简成 ax + by = 1的形式，其中a,b互质。这个问题也就变成了当a、b互质的时候，ax + by = 1总有整数解。

现在想办法对ax + by = 1求解。

上面欧几里德算法可知a、b互质则b、a%b互质，所以有

ax + by = 1

也有 bx1 + a%by1 = 1

那么可以得到：

ax + by = 1 = bx1 + a%by1 = bx1 + (a - (a/b)*b)y1

ax + by = ay1 + b(x1 - a/b*y1)

上面这个式子是恒等的，所以也就是 x = y1, y = x1 - a/b*y1。

这相当于确定了 x、y和x1、y1之间的关系。

那x1、y1又表示什么意思呢？

前面说过，x1、y1表示的是GCD(b,a%b) = 1的解。

那么整个推导串联起来就是：GCD(a,b) = 1的解x、y和GCD(b,a%b) = 1的解x1、y1有关，关系式为

x = y1, y = x1 - a/b*y1。

这条推论又可以递推下去：GCD(b,a%b) = 1的解x1、y1和GCD(a%b,b%(a%b)) = 1的解x2、y2有关，关系式为

x1 = y2, y1 = x2 - a/b*y2。

….

可以依次递推下去，那么最后一次递推应该是yn = 0,此时做边界判断。

由 GCD(k,0) = k,此时 kx + 0*y = k, 解为 x = 1 , y = 0。

我们可以得知上面推导步骤的最后边界条件应该是当yn = 0的时候，返回解 xn = 1，yn = 0，再由此逆推出x和y的解。
